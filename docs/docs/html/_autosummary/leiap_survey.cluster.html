percentiles(series)[source]¶

Returns the percentile rank for each entry

Parameters

series (Pandas series) –

Returns

Each entry in the series represents a percentile

Return type

Pandas series

Notes

This function is meant to be apply to each column (axis=1) in a dataframe using apply().It calculates the percentile score that corresponds to each observation based on the values in the entire column. So a percentile score of 23 would mean that the observation is above 23% of the scores in this column.
